واجهة إدارة العملاء والرسائل التعريفية - RKL

هذه الصفحة مخصصة لإدخال بيانات العملاء المحتملين وإدارة الرسائل التعريفية الخاصة بالمؤسسة. تم تصميمها لتعمل داخل GitHub Pages كواجهة احترافية، مع دعم حفظ البيانات محليًا، تجهيز رسالة بريد إلكتروني، وتجهيز رسالة واتساب.

إضافة / تعديل بيانات عميل

إعداد الرسالة الدعائية

ملاحظة تقنية: من داخل GitHub Pages لا يمكن إرسال واتساب أو بريد جماعي فعلي بشكل مباشر بدون خدمة خلفية. هذا النموذج يستخدم mailto: لتجهيز الإيميل، وwa.me لفتح الرسالة على واتساب. للإرسال التلقائي الحقيقي تحتاج Backend أو WhatsApp Business API أو خدمة بريد مثل Resend / SendGrid.
